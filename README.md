Título do Projeto: Portal de Notícias

Identificação: João Marcelo Monteiro de Oliveira

Descrição: Este projeto é um portal de notícias desenvolvido utilizando Node.js, Express e EJS. Ele permite a exibição de notícias dinâmicas, com funcionalidades para listar, visualizar detalhes e buscar notícias.

Como rodar: 
1. Clone o repositório
2. Instale as dependências com `npm install`
3. Crie um arquivo `.env` na raiz do projeto com a variável `API_KEY` contendo sua chave da API de notícias.
4. Inicie o servidor com `npm start`
